## Cookie

### 说明

收集网络上各位作者制作的GetCookie脚本

### 使用方法

1.点击库中的相应作者文件，点击raw获取真实地址，复制地址备用

2.Quantumult X 主界面，点击右下角风车，然后弹出界面下拉至 “重写-引用”，点击“引用”（或长按主界面“重写”图标（像支笔））

3.“引用”界面，点击右上角加号，将之前复制的地址粘贴进弹出的输入框中

4.保存并刷新
